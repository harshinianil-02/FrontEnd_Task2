Personal Portfolio Webpage

Description:-

This project is a personal portfolio webpage developed using HTML and CSS. It is used to showcase personal information, skills, and projects in a simple and structured way.

Features

- Clean and simple user interface
- Sections for About, Skills, and Projects
- Responsive design for different screen sizes
- Easy to customize

Technologies Used

- HTML
- CSS (included inside the HTML file)

File Structure

- index.html → Contains both HTML and CSS code

How to Run

1. Download the project files
2. Open the "index.html" file in any web browser

Purpose

This project is created for learning and practice of web development basics.
